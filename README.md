# Journal des décisions

## 14/07/2026 - Choix de l'API

### Décision
Utiliser TMDB comme source principale de données.

### Alternatives
- OMDb
- IMDb (non accessible directement)
- TMDB

### Justification
- API gratuite
- Documentation complète
- Grand catalogue
- Métadonnées riches
